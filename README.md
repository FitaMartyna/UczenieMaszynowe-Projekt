# UczenieMaszynowe-Projekt
Sprawozdania z przedmiotu Metody Eksploracji Danych wykonane w zespole dwuosobowym na piątym semestrze studiów na WAT

Zadania obejmują analizę i implementację następujących metod:
- dopasowanie modelu statystycznego
- funkcja logistyczna i model logitowy
- klasyfikator k-Nearest Neighbors (kNN)
- naiwny klasyfikator Bayesa
- podstawy przetwarzania języka naturalnego (NLP)

Narzędzia, które wykorzystano do realizacji:
- gretl
- Excel
- Python (scikit learn, Pandas, nltk)
